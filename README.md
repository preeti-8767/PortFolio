# Developer Portfolio Website

A modern, responsive single-page portfolio website built using HTML5 and CSS3.  
This project showcases a developer’s profile, skills, projects, and contact information with smooth navigation and attractive UI design.

---
# Features

- ✅ Responsive Single Page Design
- ✅ Smooth Scroll Navigation
- ✅ Glassmorphism Navbar
- ✅ Modern Gradient UI
- ✅ Skills Progress Bars
- ✅ Project Hover Effects
- ✅ Clean & Professional Layout
- ✅ Mobile Friendly Design

---

# Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
